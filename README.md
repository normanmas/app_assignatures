# App Assignatures
Aplicació web, desenvolupada amb suport de la IA, per tenir una llista actualitzada de les diferents assignatures del grau d'ingenieria informàtica de la UOC.  
Permet consultar , importar (web scrappin), i comparar assignatures entre les diferents carreres.  
## Objectiu  
Aquest projectes serveix per aprendre noves funcions amb Python mentre es trien les assignatures.  
## Funcionalitats actuals:
- Aplicació web amb Flask.
- Llistat d'assignatures en una taula.
- Base de dates de SQLite.
- Importació d'assignatures des d'una pàgina de grau de la UOC.
- Lectura del pla docent amb Selenium.
- Actualització del model d'Avaluació i la descripció d'una assignatura.
- Enllaç directe al pla docent original.

## Tecnologies
- Python
- Flask
- SQLite
- BeautifulSoup
- Requests
- Selenium
- HTML
- CSS

Activa l'entorn virtual:  
```bash
source entorn/bin/activate
```
Instal·la dependències:  
```bash
pip install flask requests beautifulsoup4 selenium
```
Executa l'aplicació:  
```bash
python app.py
```
Obre al navegador:
```bash
http://127.0.0.1:5000
```
